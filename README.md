cd hphrms_api
cp .env.example .env
php artisan migrate
php artisan db:seed

> **Note:** The implementation is framework-lightweight so it can run in restricted environments without Composer. All classes are namespaced under `App\` and auto-loaded via `vendor/autoload.php`.
